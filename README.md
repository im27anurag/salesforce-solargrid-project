# ☀️ SolarGrid CRM & Support System

A Salesforce-based CRM and customer support solution designed for managing solar installation sales, customer inquiries, support operations, and self-service knowledge management.

This project demonstrates end-to-end implementation using Sales Cloud, Service Cloud, automation tools, dashboards, data management, and Experience Cloud concepts.

---

# 🚀 Project Overview

SolarGrid required a centralized platform to:

- Manage incoming solar installation leads
- Track sales opportunities and revenue pipeline
- Automate customer support processes
- Handle support cases through Email-to-Case
- Provide self-service knowledge resources
- Improve operational visibility using reports and dashboards

This implementation was built using Salesforce Sales Cloud and Service Cloud with automation and customer support workflows.
